RPB components developed by DKTK:

  • ddRPB -- radiotherapy clinical research IT infrastructure (GNU GPL):
    • ddRPB-core -- Core library
    • ddRPB-web-API -- Unified platform web-API
    • ddRPB-portal -- Unified platform portal
    • ddRPB-mobile -- Patient Reported Outcomes (ePRO) and Clinician Outcome Assessment (eCOA)
    • Randomisation (inspired by RANDI) -- Randomization process in clinical trials
  • ddRPB-server -- Program access to the platform functionality - web services (GNU GPL)
  • ddRPB-client -- Application for DICOM data de/re-identification and upload/download (GNU GPL)
RPB integrates with several existing OSS, each with its own source license:
  • EDC - OpenClinica -- Electronic Data Capture (GNU LGPL)
  • PACS - Conquest -- DICOM Server (PUBLIC DOMAIN)
  • LAB - LabKey -- Laboratory Data Management (ALv2)
  • PIDG - Mainzelliste -- Patient PID Generator (pseudonymisation) (AGPL)
  • DICOM Viewers:
    • WEASIS -- Java WebStart (EPL)
  • HL7 - NextGen Connect-- Health care integration engine (MPL 1.1)
  • XForms - Enketo -- Web form rendering engine (OpenRosa API support) for ePRO/eCOA (ASL)
  • CTP - Clinical Trials Processor -- Image processing application for imaging clinical trials data (RSNA Public License)
