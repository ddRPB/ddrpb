RadPlanBio components developed by DKTK Dresden site:

  • RPB-Portal -- Unified access to the platform functionality (GNU GPL)
    • Randomisation - inspired by RANDI -- Randomization process in clinical trials (GNU GPL)
  • RPB-Web-API -- Programatic access to the platform functionality - web serivices (GNU GPL)
  • RPB-Desktop-Client -- Application for DICOM data de/re-identification and upload/download (GNU GPL)
RadPlanBio integrates several existing software systems, each with its own source license:
  • EDC - OpenClinica -- Electronic Data Capture solution for collection of study data (GNU LGPL)
  • PACS - ConQuest -- DICOM PACS server as a medical image data storage (PUBLIC DOMAIN)
  • PIDG - MainzelListe -- patient PID generator (pseudonymisation) (AGPL)
  • DICOM viewers:
    • DWV -- HTML5 (GNU GPL)
    • WEASIS -- JAVA WebStart (EPL)
